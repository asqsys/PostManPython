# DIFF_PYTHON

## Installation

Use the python to create a virtual enviroment.

```bash
python -m venv myapp
```

After the virtualenv created, clone the source code from git inside a folder in virtualenv.

Activate the virutalenv, using the activate.bat inside Scripts/activate.bat.

Install the requirements using the package manager [pip](https://pip.pypa.io/en/stable/)

```bash
pip install -r src/requirements.txt
```

To run:
```bash
python app.py
```
